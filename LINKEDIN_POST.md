# LinkedIn Post

I built **Hand Focus Fruit Ninja**, a browser-based game controlled entirely with hand gestures.

Players slice fruit using their index finger while the app uses real-time hand tracking through the device camera. At the end of the round, the game shows a privacy-focused two-column reveal explaining how camera permission works and why users should pay attention to what they allow.

Tech used:

- JavaScript
- Vite
- MediaPipe hand tracking
- Browser camera APIs
- Canvas rendering

This project was a fun mix of gameplay, computer vision, responsive UI, and privacy awareness.
